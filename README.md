

 #  PROJECT OVER VIEW

 Zecpath AI System is a modular AI recruitment pipeline 
designed for resume parsing, ATS scoring, candidate screening, 
and interview preparation.

# PROJEXT STRUCTURE

Project Structure:

data/              - Dataset storage
parsers/           - Resume parsing logic
ats_engine/        - ATS scoring engine
screening_ai/      - Screening logic
interview_ai/      - Interview module
scoring/           - Candidate scoring
utils/             - Utility functions (logging)
tests/             - Unit tests
main.py            - Entry point
requirements.txt   - Dependencies

## Setup Instructions

1. Clone repository
2. Create virtual environment:
   python -m venv venv

3. Activate environment:
   venv\Scripts\activate

4. Install dependencies:
   pip install -r requirements.txt
